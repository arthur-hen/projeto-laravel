
# ✅ **REQUISITOS QUE PRECISAM ESTAR INSTALADOS NO PC**

Antes de qualquer comando, a pessoa precisa ter instalado:

### ✔ PHP 8.2)

### ✔ Composer

### ✔ XAMPP


### Criar o projeto Laravel


composer create-project laravel/laravel site


### Entrar na pasta do projeto


cd site


### Gerar a key do Laravel


php artisan key:generate



# — INSTALAR DEPENDÊNCIAS



composer require laravel/breeze --dev

php artisan breeze:install

npm install
npm run build



# MIGRATIONS


php artisan migrate


# SUBIR O SERVIDOR LOCAL

php artisan serve


A aplicação abre em:

http://127.0.0.1:8000/carros
